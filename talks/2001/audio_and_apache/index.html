ApacheCon 2001: Audio and Apache Talk

Update (11/25/2002): I recently gave a revised version of this talk with updated notes.

On April 6th, 2001 in Santa Clara, California. I did a talk about Audio and Apache. The Session Audio and Apache had Notes and Slides that I am providing here.

Day: Friday, 4:00pm.
Duration: 60 minutes
Session chair: Craig McClanahan
Style: Presentation
Level: Experienced
Audience: Technical
Room: 201
Categories: Apache HTTP Server, New Technology, Performance
Speaker: Sander van Zoest, Covalent Technologies, Inc.
Abstract:

It is time to explore the world of serving audio using the Apache HTTP server. This session will discuss streaming, on-demand audio, configuration of modules, mime types, audio formats, etc.
